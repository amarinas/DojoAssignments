What is Git?

git is a version control system; you can think of it as a series of snapshots(commits) of your code base. With the creation of each snapshot you build out a whole application, seeing the growth of a full app. durring this time you can make branches to experiment and come to a previous sna shot of your took or merge those experiments into the your app.

What is GitHub?

GitHub is a code hosting platform fro version control and collaoration. it let you and others work on projects from anywhere using those code snapshots that you took during development
